### What Went Well

- Verification discipline improved across the epic: each security/privacy story closed with deterministic scripts and full suite validation.
- Review-to-remediation loops were fast and specific, with minimal collateral churn.
- Feature boundaries remained stable while adding deeper compliance constraints.
- Privacy posture became enforceable via code + checks, not documentation-only intent.

### Challenges and Friction

- Security/compliance stories required multiple rounds to harden fail-closed behavior and migration semantics.
- Static string-based verifier checks were initially insufficient for gate behavior and needed runtime semantic assertions.
- Several concerns surfaced only during adversarial review, reinforcing the need for mandatory CR passes before done.

### Cross-Story Patterns

1. Fail-closed first, then optimize.
- Any path touching authorization, sensitive persistence, or feature gates should default to denial on uncertainty.

2. Guardrail scripts must test behavior, not only source text.
- Runtime semantics and deterministic self-tests catch bypasses that regex checks miss.

3. Migration safety requires preserve-then-cleanup sequencing.
- Data-preserving migration before deletion was a recurring correctness principle.

4. Parent-protected boundaries need consistent enforcement at entry points.
- Uniform child-mode denial with safe messaging reduced policy drift.

### Technical Debt and Risks

- Medium: Phase 2-enabled gate branch is intentionally unreachable in Phase 1 and will need dedicated tests when enablement is introduced.
- Low: Guardrail script coverage is broad but still heavily file-shape dependent; future refactors should preserve semantic checks.

### Key Lessons Learned

- Compliance features are production features: they need first-class contracts, deterministic tests, and adversarial review.
- "Passing checks" is not enough unless checks validate runtime decision semantics.
- Closing stories only after follow-up CR clean status materially improved reliability and confidence.
